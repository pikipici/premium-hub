# PremiumHub Web (Next.js)

## Quick Start

```bash
cp .env.example .env.local
npm install
npm run dev
```

## Env

- `NEXT_PUBLIC_API_URL` -> endpoint backend (`/api/v1`)
- `NEXT_PUBLIC_GOOGLE_CLIENT_ID` -> Google OAuth Client ID (browser)

Google button otomatis tampil kalau `NEXT_PUBLIC_GOOGLE_CLIENT_ID` diisi.
